## Content clusters (future blog/docs)

1. **Website Security** — headers, scores, common misconfigurations
2. **SSL Certificates** — expiry, renewal, HTTPS redirects
3. **Website Monitoring** — uptime vs security monitoring
4. **Website Changes** — unauthorized scripts, defacement signals
5. **Security Headers** — HSTS, CSP, X-Frame-Options explainers
6. **Small Business Cybersecurity** — practical guides
7. **Agency Monitoring** — multi-client workflows, reporting

## Internal linking strategy

- Homepage → all feature pages, `/scan`, `/pricing`, `/signup`
- Feature pages → related features + conversion CTAs
- Pricing → feature pages for intent match
- Trust pages (`/about`, `/security`, `/contact`) → homepage + pricing
- Footer on every marketing page → features, legal, contact

## Conversion paths (every SEO page)

Each landing page includes: **Scan** · **Pricing** · **Signup** (and **Demo** on enterprise pages).

## AI search optimization

- Consistent entity: **CyberShield Cloud** — website security monitoring software
- FAQ schema on homepage
- Structured data: Organization, WebSite, SoftwareApplication, Product, Breadcrumb
- Clear H1/H2 hierarchy on feature pages
- Unique copy per URL (no duplicate feature blurbs)
